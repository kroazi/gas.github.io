指令 参数 描述
登录
/register 密码 确认密码 注册
/login 密码 登录
/changepassword 当前密码 新密码 修改密码
/unregister 密码 注销账户 
传送
/tpa 玩家 将我传送到玩家那里
/tphere 玩家 将玩家传送到我这里
/tpaccept 无 同意传送请求
/tpdeny 无 拒绝传送请求
家
/sethome 名称 在当前位置设置家
/home 名称/无 将你传送到⼀个家/列出你的家
/delhome 名称 删除一个家
私聊
/msg 玩家 信息 向玩家发送私信
/r 信息 快速回复上⼀位给你发消息的玩家
皮肤
/skins 无 打开皮肤GUI菜单
/skin 玩家 将玩家的皮肤变成你的皮肤
/skin clear 重置您的皮肤
/skin update 更新皮肤
其他
/suicide 无 自杀
/skull 玩家 获取玩家头颅
/hat 无 把手中的物品戴在头上
/disenchant 无 移除手持物附魔效果
/disposal 无 带GUI的垃圾桶(比自带的快)
/lag 无 向您显⽰有关您的服务器的⼀些性能数据
/rules 无 显⽰服务器规则
/sign 内容 签署您当前持有的物品
/unsign 无 删除您当前持有的项⽬的签名
/co i 查看方块交互历史
